# Корпоративная лента новостей (Spring Boot + Angular)

### Задание:
Лента корпоративных новостей. Разработан бэкенд на Java Spring Boot (сущность News: title, text, publicationDate) со встроенными тестовыми данными. Фронтенд на Angular использует карточный дизайн, вывод с сортировкой по дате через Date Pipe, переход на отдельную страницу (Роутинг) и публикацию через Reactive Forms.

### Команда для старта всей системы в Docker:
```bash
docker compose up -d --build

---

### Как теперь всё запустить и проверить для видео:
1. Откройте терминал в корне проекта `corporate-news`.
2. Запустите команду: `docker compose up -d --build`
3. Контейнеры соберутся и запустятся. 
4. Откройте `http://localhost:4200` — вы увидите рабочий карточный интерфейс, уже заполненный тремя тестовыми новостями, которые мы прописали в коде Java-репозитория!
